# results/

Top-level results directory. Paper-specific results have moved to
`use_cases/IndPenSim/results/`.

- `figures/` — figures referenced by the AI4D 2026 paper. Mirrors of these
  PNGs (with the scripts that produced them) live in
  `use_cases/IndPenSim/figures/`.
- `expA_*.json`, `expB_*.json` — the JSON outputs of the paper experiments.
  Authoritative copies are in `use_cases/IndPenSim/results/`.
- `legacy/` — older result files from earlier iterations of the project,
  kept for reference. Not used by the paper or the package.
